# Figures

The **Figures** section provides focused, tradition-specific profiles for each major messianic, mediator, or restorer figure covered on this site. Pages document attested sources, dating, key motifs, and where each figure sits within the broader comparative framework.

Figures are organized by tradition, roughly chronological from deepest antiquity. For how they relate to each other and to the site's five research zones, see the [site introduction](/docs/intro) and the [Messianic archetypes overview](../overview/messianic-archetypes.md).
