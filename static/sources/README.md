# Data files (`static/sources/`)

CSV tables extracted from source workbooks, one file per sheet.
Paths are served as `/sources/<filename>.csv`.

## Naming convention

`<workbook-prefix>-<sheet-slug>.csv`

| Prefix | Source workbook |
| --- | --- |
| `christodex-` | Kosmographica – Christodex (Alpha) |
| `daimonologia-` | Interpretatio Universalis – Daimonologia |
| `greek-phil-` | Greek Philosophers List |
| `kindex-` | Kosmographica Indexes BETA |
| `kvol2-` | Kosmographica Vol 2 |
